# Inbox for Agent Zero

This folder holds **improvement requests from the human/collaborator** for Agent Zero.

- **IMPROVEMENTS_FROM_HUMAN.md** — Main file. Add dated tasks and priorities; the agent is instructed to read it when doing self-improvement or upgrades.
- Push changes to the **self** remote so the agent sees them:  
  `./scripts/setup/git-self-admin.sh push self main` (uses `GITHUB_TOKEN`).

See [Private Repo for Agent Zero](../../docs/setup/PRIVATE_REPO_AGENT_CHANGES.md) for the full two-way workflow.
