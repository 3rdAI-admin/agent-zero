## Git remote "self" (Agent Zero's repo — self-improvement, upgrades, updates)

The remote **self** is **Agent Zero's own private repository** (e.g. 3rdAI-bill/agent-zero-private-repo): use it for self-improvement, upgrades, and updates. For any git operation that targets **self** (push, pull, fetch), use the helper script so the self-improvement token is used: `./scripts/setup/git-self.sh` then the git args. Example: `./scripts/setup/git-self.sh push self main`. Do not run raw `git push self` or `git pull self`; use the script.

**Improvement requests from human:** Before or when doing self-improvement or upgrades, pull from **self** and read **inbox/IMPROVEMENTS_FROM_HUMAN.md**. Treat listed items as requested work; implement or schedule them as appropriate.
