# Testing Scripts

Scripts for testing and validating Agent Zero functionality.

## Available Scripts

- **validate.sh** - Comprehensive validation script
- **test_claude_integration.sh** - Test Claude Code integration
- **test_claude_mcp_simple.sh** - Simple MCP connection test
- **test_integration_complete.sh** - Complete integration test
- **test_mcp_connection.py** - Python MCP connection test
- **test_server.py** - Server testing script
- **test_burp_scan.sh** - Burp Suite scan test

## Usage

```bash
# Run full validation
./scripts/testing/validate.sh

# Test Claude integration
./scripts/testing/test_claude_integration.sh

# Test MCP connection
./scripts/testing/test_claude_mcp_simple.sh
```
