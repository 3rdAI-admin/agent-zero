# Documentation

This directory contains all Agent Zero documentation organized by category.

## Structure

- **guides/** - Setup guides, quick starts, and how-to documentation
- **integration/** - Integration guides and examples
- **troubleshooting/** - Troubleshooting guides and fixes
- **archive/** - Historical and obsolete documentation
- **res/** - Images and assets for documentation

## Contributing

When adding new documentation:
1. Place setup guides in `guides/`
2. Place integration docs in `integration/`
3. Place troubleshooting in `troubleshooting/`
4. Move obsolete docs to `archive/`
